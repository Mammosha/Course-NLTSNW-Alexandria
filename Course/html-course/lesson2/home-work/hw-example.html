Що таке CMS(Content Managment System)

CMS - система управління контентом (движок) - написана PHP-програмістами основа для сайту, за допомогою якої ви зможете керувати сайтом (додавати контент, змінювати пункти меню, тощо ) не знаючи HTML та CSS.

Однак, для того, щоб зробити сайт за допомогою CMS, будуть потрібні послуги і програміста, і дизайнера, і верстальника.

Які бувають CMS

Бувають різні системи управління контентом: для інтернет магазинів, блогів, форумів...
